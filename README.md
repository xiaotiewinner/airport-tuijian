## 四、 拿到账号后怎么用？(Windows/手机 通用逻辑)

购买成功后，服务通常会立即开通。以下教你如何获取配置并连接。

### 第一步：获取订阅链接 (Subscription Link)
1.  登录 Just My Socks 官网。
2.  点击菜单栏的 **Services** -> **My Services**。
3.  点击你刚刚购买的那个服务（状态显示为 Active）。
4.  在详情页中，你会看到一堆服务器信息。请找到 **Subscription link** 这一栏。
5.  这也是最方便的配置方式：点击链接旁边的 **Copy** 按钮，复制以 `https://` 开头的订阅地址。

### 第二步：安装客户端
你需要下载支持 V2Ray/Shadowsocks 协议的客户端。
*   **Windows**: 推荐 **v2rayN**
*   **Android**: 推荐 **v2rayNG**
*   **iOS**: 推荐 **Shadowrocket** (小火箭，需美区Apple ID购买)
*   **Mac**: 推荐 **V2RayU** 或 **Foxray**

*(客户端下载地址请参考 GitHub 或相关软件站，本文以 Windows 版 v2rayN 为例)*

### 第三步：配置 v2rayN (图文简述)

1.  **下载与解压**：下载 v2rayN 压缩包并解压，双击运行 `v2rayN.exe`。
2.  **添加订阅**：
    *   打开软件主界面，点击顶部菜单的 **“订阅分组”** -> **“订阅设置”**。
    *   点击 **“添加”**。
    *   **备注**：随便填，例如 `JMS`。
    *   **可选地址(url)**：**粘贴你在第一步复制的订阅链接**。
    *   点击“确定”。
3.  **更新订阅**：
    *   回到主界面，点击顶部菜单的 **“订阅分组”** -> **“更新当前订阅(不通过代理)”**。
    *   此时，软件下方列表会出现 3-6 个服务器节点（通常命名为 c1, c2, c3...）。
4.  **连接节点**：
    *   鼠标左键选中一个节点（建议选 `s3` 结尾的，通常是 CN2 GIA 线路）。
    *   按回车键（Enter）激活该节点（变为蓝色或打钩）。
5.  **开启代理**：
    *   在电脑右下角任务栏找到 v2rayN 的小图标（蓝色V字）。
    *   **右键点击**图标 -> **系统代理** -> 选择 **“自动配置系统代理”**。
    *   图标变红，代表代理已开启。

### 第四步：测试
打开浏览器，访问一下 Google 或 YouTube。如果能秒开，恭喜你，你已经拥有了自由畅快的网络体验！🎉
